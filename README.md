# LLM-Extract

Extract structured data from text content using LLMs.

## Installation

At this stage the package is not distributed, ie not published to PyPI. To install it, clone the repository and install it with pip: `pip install .`
Or use `pip install -e .` if you want to install in editable mode.

## Usage

To run the example in `data/examples`, use the command: `llm-extract --configfile=data/examples/config.yaml`
